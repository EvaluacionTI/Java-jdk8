Ejemplo de un documento HTML

El cuerpo del documento puede contener:

Texto, imágenes, sonido y órdenes HTML

HTML es el lenguaje utilizado para desarrolar páginas y documetos WEB

A diferencia de los lenguajes conversionales, HTML utiliza una serie de etiquetas ASCII especiales intercaladas en un documento escrito en ASCII
Dichas etiquetas serán posteriormente interpretadas por los exploradores encargados de visualizar la página o el documento WEB, con el fin de establecer el formato Mediante el HTML se puede obtener una salida de información al Cliente que le permite una mejor y tomar una acción sobre él.
